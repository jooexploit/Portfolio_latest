Islamic Shoky

A productivity-focused VS Code extension built for Muslim developers, integrating location-based prayer notifications, rotating Azkar prompts, and a focus task manager.

1. Recruiter-Friendly Summary

Developers frequently spend uninterrupted hours coding, losing track of break times, daily Azkar, and local prayer times, affecting productivity and well-being.

2. Platform Features &amp; Architecture

Built a native VS Code extension displaying prayer times directly in the editor's status bar. It includes Pomodoro timers, task lists, and unobtrusive reminders calculated purely offline.

3. Engineering Challenges

Built with VS Code Extension APIs. It uses coordinate calculations for prayer schedules, avoiding external HTTP requests, and persists state in workspace storage.

4. Implemented Solutions

Created an integrated workflow tool that lets developers stay mindful of their faith routines directly from their workspace.
