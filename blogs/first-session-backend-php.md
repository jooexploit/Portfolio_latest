# First Session Backend PHP
بسـم اللـه الرحمـن الرحيـم الحمد لله رب العالمين، والصلاة والسلام على سيدنا محمد الصادق الوعد الأمين، اللهم لا علم لنا إلا ما علمتنا، إنك أنت العليم الحكيم، اللهم علمنا ما ينفعنا، وانفعنا بما علمتنا وزدنا علماً  
  
سلسله تانيه بس المره دي backend مع [EraaSoft](https://www.linkedin.com/company/eraasoft/) ف البدايه البشمهندس [Islam kabbary](https://www.linkedin.com/in/islam-kabbary-2b35a814b/) كان بدأ بيعني اي اصلا backend و اي الفرق بينه وبين frontend ف بص بقي يا سيدي وعايزك كد تتخيل معايا اننا فاتحين صفحه انشاء حساب اول اما فتحت شفت قدامك خانات عشان تكتب بياناتك ف الي انت شايف دا frontend امامي حاجه انت شايفها طب انت خلاص كتبت البيانات ودوست تأكيد ف يجي بقي دور backend تقوم واخده بيانات وتشوف انت جي منين ومعاك اي وعلي الاساس دا تتعامل هي ف هي لقت انك جي من تسجيل الدخول ف تروح بعته بقي لل DB تقلها سجلي الراجل دا عندك (طبعا انت الي بتبقي معرفه تسجله فين وتسجل اي) دا بشكل مبسط الفرق بينهم تعالا بقي نركز علي ال backend المره دي  
  
بص يسيدي هو ليه لغات كتيير وكل لغه متميزه فحاجه وكل حاجه ليها ميزه وليها برضه عيب بس احنا انهارده هنركز علي لغه php بس قبل اما نبدأ فيها عايزك افهمك فرق بين حاجتين مهمين جداا وهم ال client side , server side تعالا نبدأ ب client side  
  
دي يسيدي زي اللغات html,css كد انت ممكن تخش علي inspect وتعدل الكود وتشوف كل حاجه عادي طب انت من وجهه نظرك هل دا ينفع للbackend انه يمشي بنفس النظام والكود الي مهم جداا وبيبقي فيه حاجات مهمه انه يبقي قدام الناس كلها ويقدروا يشوفوه ف لا طبعا مينفعش ف عشان كد ال backend شغال ب server side ودا عباره عن انك بتبعت للسيرفر تقله والله 5+5 يساوي كام كمثال يعني فهو يشوف عنده ويحسب بعد كد يرد عليك انت متعرفش هو حسبها ازاي ولا شفت اي الي حصل بس هو خلص وبعتلك الي انت عايزه ف الاخر ف دا الفرق بينهم ببساطه يعني تعالا بقي يسيدي نخش علي ال php  
  
انت كد فهمت انها عشان تعرف تتكلم معاها لازم سيرفر ف احنا بنقوم سيرفر بسيط اسمه apache هتحمل برنامج بسيط زي (lampp,xampp) وتشغل بقي السيرفر بس السيرفر دا بيبقي ليه مكان بس بيقرأ منه ف انت شوف البرنامج بتاعك انت سطبته فين وهتلاقي مجلد مثلا اسمه اما (htdocs/www) وهتبدأ تعمل فيه بقي ملف (index.php) اول حاجه معانا ف php وهي  
  
```  
  
php?>  
  
```  
  
ممكن تتقفل وممكن لا لو هتكتب جواها html بتقفلها لو لا يبقي عادي هو هيعرف ان كل حاجه php نخش علي الي بعده وهو امرين مهمين  
  
```  
;"echo "hi  
  
;"print " hi  
  
```  
وهنا الاتنين دول زي بعض بس الفرق بسيط وهو ان echo بيبقي ليها اكتر من حاجه ممكن تتطبعهم ف نفس الوقت مش شرط يبقي hi فقط اما print ف لا مفيهاش كد  
نخش علي الموضوع الي بعده وهو كان datatype in php  
```  
<?php  
$name = "jooexploit";   // string (نص)  
$age = 20;        // integer (عدد صحيح)

$colors = array("red", "blue", "green"); // array (مجموعة بيانات)  
```
